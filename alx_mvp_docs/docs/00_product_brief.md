# Product Brief

## Nome provisório
Auto Dealer MVP

## Tipo de produto
Plataforma web para revenda/estoque automotivo com catálogo público, páginas de conversão e painel administrativo interno.

## Objetivo do MVP
Criar um protótipo navegável, operável e apresentável a um cliente potencial, demonstrando a capacidade de:
- publicar estoque de veículos
- gerar leads qualificados
- administrar veículos e contatos
- oferecer base técnica pronta para evolução futura

## Problema que o produto resolve
Muitas revendas pequenas e médias operam com sites desatualizados, pouca integração comercial, baixa clareza de estoque e fraco rastreamento de leads. O MVP deve mostrar uma alternativa moderna, rápida, administrável e preparada para crescimento.

## Resultado esperado
Ao final do MVP, o cliente deve visualizar:
- um site moderno com estoque realista
- um fluxo claro de geração de leads
- um admin simples e funcional
- uma base de software confiável para evolução em fases futuras

## Público-alvo
### Primário
- donos de revendas
- gerentes comerciais
- gestores de marketing automotivo

### Secundário
- vendedores
- responsáveis por cadastro de estoque
- equipe administrativa

## Proposta de valor
- catálogo moderno e rápido
- experiência mobile forte
- foco em conversão por WhatsApp e formulários
- admin simples para operação do dia a dia
- stack atual, manutenção facilitada e boa escalabilidade

## Benchmark de referência
Usar como referência funcional geral plataformas de estoque automotivo com:
- home comercial
- listagem filtrável
- detalhe do veículo
- formulários de contato/financiamento/venda
- páginas institucionais

## Restrições
- não copiar código
- não copiar textos
- não copiar assets visuais
- não copiar identidade de marca
- não replicar UX de forma servil

## Meta de entrega do protótipo
- deploy em ambiente público
- seed com conteúdo de demonstração
- fluxos centrais operantes
- admin funcional
- roteiro de demo pronto
