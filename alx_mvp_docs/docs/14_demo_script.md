# Demo Script

## Objetivo da apresentação
Mostrar ao cliente que a solução já existe em forma de protótipo funcional, cobrindo os fluxos essenciais do negócio.

## Ordem recomendada
1. Home
2. Estoque
3. Detalhe do veículo
4. Geração de lead
5. Admin
6. Evolução futura

## Roteiro
### 1. Home
Apresentar:
- proposta visual
- CTA principal
- destaques do estoque
- blocos de financiamento e venda

### 2. Estoque
Apresentar:
- filtros
- velocidade
- usabilidade mobile/desktop
- cards consistentes

### 3. Detalhe
Apresentar:
- galeria
- ficha técnica
- opcionais
- CTA de contato/WhatsApp

### 4. Lead
Executar um envio real de formulário.
Depois abrir o admin e mostrar o lead salvo.

### 5. Admin
Mostrar:
- criação/edição de veículo
- publicação/despublicação
- gestão de leads
- edição de conteúdo

### 6. Próximos passos
Encerrar com:
- integrações possíveis
- CRM avançado
- SEO
- automações
- portal do consignante
