# Feature Scope

## Escopo do MVP

### Público
1. Home comercial
2. Página de estoque
3. Página de detalhe do veículo
4. Página de contato
5. Página de financiamento
6. Página de vender veículo
7. Página institucional “quem somos”
8. Blog/notícias simples
9. Política de privacidade / termos

### Conversão
1. CTA para WhatsApp em home, listagem e detalhe
2. Formulário de contato geral
3. Formulário de interesse em veículo
4. Formulário de financiamento
5. Formulário de venda de veículo
6. Captcha em formulários públicos

### Admin
1. Login admin
2. Dashboard simples
3. CRUD de veículos
4. Upload e ordenação de imagens
5. Gestão de leads
6. CRUD de páginas institucionais
7. CRUD de posts simples
8. Configurações básicas do site

### Observabilidade mínima
1. Analytics
2. Captura de erros
3. Logs básicos de lead

## Fora do escopo do MVP
- checkout
- pagamento online
- pedidos
- assinatura
- marketplace multi-loja
- app mobile nativo
- CRM avançado com automações complexas
- scoring avançado com IA
- simulador financeiro sofisticado
- integração com Detran/FIPE/ERP/financeiras reais
- comparação avançada de veículos
- favoritos com conta pública obrigatória
- catálogo multi-tenant
- chat em tempo real

## Fase 2 sugerida

- múltiplos perfis admin
